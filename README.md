#  Calendar Service

##  Описание

**Calendar Service** — сервис доменной логики FocusFlow, отвечающий за работу с календарями, событиями, заметками и статусами дней.  
Он обеспечивает создание и управление контентом календаря, взаимодействует с Category Service для проверки категорий и формирует данные для последующей синхронизации.

---

##  Функционал

- Создание, редактирование и удаление событий  
- Получение событий по диапазону дат  
- Управление заметками  
- Изменение статуса дня (рабочий/выходной)  
- Проверка категорий через Category Service  
- Подготовка данных для синхронизации (через абстракцию MessageBus)
